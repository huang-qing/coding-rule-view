# codingRuleView-编码规则展示

#### 参数

+   类型：数组
+   单个对象结构

        {
            type:"01", 
            length:5, 
            name:"编码规则文字解释" 
        }


#### type(编码类型):
+ "01": 数字类型
+ "02": 字母类型
+ "03": 数字|字母类型
+ "04": 流水码类型
+ "05": 日期类型

#### length(编码长度): 
有效值为0到正无穷大。长度为0时，表示任意长度。

#### name(编码规则文字解释):
编码规则的文字描述
